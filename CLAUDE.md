# 마을 디펜스 - 로그라이크 게임

HTML5 Canvas 기반 탑다운 로그라이크 마을 디펜스 게임

## 프로젝트 구조

```
game01/
├── index.html    # 단일 파일 게임 (HTML + CSS + JS)
└── CLAUDE.md     # 프로젝트 문서
```

## 게임 개요

- **장르**: 탑다운 로그라이크 마을 디펜스
- **목표**: 마을 중심부(HP 10,000)를 몬스터로부터 방어
- **패배 조건**: 플레이어 사망 또는 마을 파괴

## 직업 시스템

### 전사 (Warrior)
- **타입**: 근접 딜러 / 탱커
- **스탯**: 체력 120, 공격력 25, 속도 4
- **공격**: 근접 (검)
- **스킬**:
  - Q: 돌진 - 전방 돌격 + 200% 데미지 + 무적 (5초)
  - W: 전쟁의 함성 - 주변 적 기절 + 공격력 50% 버프 (8초)
  - E: 회전 베기 - 360도 지속 피해 (6초)
  - R: 대지 분쇄 - 광역 500% 데미지 + 기절 (30초)

### 마법사 (Mage)
- **타입**: 원거리 딜러
- **스탯**: 체력 70, 공격력 35, 속도 3.5
- **공격**: 원거리 (마법 투사체)
- **스킬**:
  - Q: 파이어볼 - 폭발 마법 300% + 범위 폭발 (4초)
  - W: 텔레포트 - 순간이동 + 무적 (6초)
  - E: 블리자드 - 범위 슬로우 + 지속 피해 (10초)
  - R: 메테오 - 8개 메테오 낙하 400% 데미지 (30초)

### 궁수 (Archer)
- **타입**: 원거리 딜러 / 기동
- **스탯**: 체력 85, 공격력 20, 속도 5
- **공격**: 원거리 (화살, 빠른 공격속도)
- **스킬**:
  - Q: 관통화살 - 적 관통 250% 데미지 (3초)
  - W: 구르기 - 회피 이동 + 무적 (4초)
  - E: 다중사격 - 5초간 3발씩 발사 (7초)
  - R: 화살비 - 범위 지속 피해 (25초)

## 조작법

| 키 | 동작 |
|---|------|
| 방향키 | 이동 + 바라보는 방향 설정 |
| Space | 기본 공격 (근접/원거리) |
| Q/W/E/R | 스킬 |
| M | 배경음악 ON/OFF |
| Enter | 상점 닫기 |

## 구현된 기능

### 플레이어 시스템
- 직업 선택 (게임 시작 시)
- 직업별 다른 스탯, 공격 방식, 스킬
- 근접/원거리 공격 분기
- 버프 시스템

### 몬스터 시스템
- 일반 몬스터 (빨간색)
- 엘리트 몬스터 (보라색, 웨이브 3+)
- 보스 (5웨이브마다)
- AI: 플레이어 우선, 마을 공격
- 기절/슬로우 상태

### 아이템 드랍
| 아이템 | 효과 | 드랍률 |
|-------|------|--------|
| ❤️ 체력 포션 | HP +30 | 15% |
| 💰 골드 주머니 | 골드 획득 | 25% |
| ⚔️ 분노의 물약 | 공격력 +50% (8초) | 8% |
| 💨 신속의 물약 | 이동속도 +50% (8초) | 8% |
| ⏳ 시간의 모래 | 스킬 쿨다운 초기화 | 3% |
| 🛡️ 보호막 | 50HP 보호막 | 5% |

### 상점 시스템
- 공격력 / 이동속도 / 체력 / 공격범위 업그레이드
- 체력 회복, 마을 수리

### 오디오
- 8비트 배경음악 (일반/보스 테마)
- 효과음 (공격, 피격, 스킬, 아이템)

### 마을
- 중심부 (수정탑): HP 10,000
- 건물 6개 + NPC 5명
- 동서남북 경로 스폰
